Pro Tutor Landing Page

A responsive landing page for Pro Tutor, offering programming courses.

Technologies Used:

- HTML5
- CSS3
- JavaScript (ES6+)
- Owl Carousel (via CDNJS) for sliding effects
- Font Awesome for icons and styling

Project Structure:

The project is divided into sections:

1. Home: Introduction to Pro Tutor and its offerings
2. Categories: List of programming categories and courses
3. Courses: Detailed course information and descriptions
4. Testimonials: Student testimonials and reviews
5. Services: Additional services offered by Pro Tutor

Features:

- Responsive design for optimal viewing on various devices
- Smooth sliding effects using Owl Carousel
- Interactive elements using JavaScript
- Font Awesome icons for visually appealing styling
